# Developer Notes

## ℹ️ Overview

This repository contains **curated developer notes and cheat sheets** focused on concepts, tools, and workflows that have remained stable over time.

It is not a tutorial collection or a comprehensive knowledge base. Everything here exists because it has proven useful repeatedly in real work.

## 📦 Scope and Intent

The goal of this repository is to document:

- Fundamentals that don't change often
- Commands and patterns I actually use
- Concepts worth remembering without re-deriving every time

This is intentionally:

- Incomplete
- Opinionated
- Practical

## 📁 Project Structure

### cheat-sheets/

Short, annotated references for tools and topics I regularly reach for.

Each cheat sheet:

- Focuses on day-to-day usage
- Includes brief context or comments
- Avoids exhaustive flag listings or vendor docs
